Sharkey ActivityPub Extensions

These are the extensions for ActivityPub that Sharkey implements, this page uses compact IRIs.

  • The ap prefix is used to refer to https://www.w3.org/ns/activitystreams#
  • The sharkey prefix is used to refer to https://joinsharkey.org/ns#

backgroundUrl

  • Compact IRI: sharkey:backgroundUrl
  • Canonical IRI: https://joinsharkey.org/ns#backgroundUrl

Used on actors to indicate that they have a background image url expressed as string that can be used to give additional customizations to their profiles. If set to undefined or null it will be ignored
